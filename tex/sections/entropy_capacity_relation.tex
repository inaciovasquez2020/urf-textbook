\section{Capacity--Entropy Relation}

Define the feasibility functional
\[
\Phi(x) = H(x) - C .
\]

\begin{theorem}[Capacity--Entropy Constraint]
For any admissible system configuration \(x\), feasibility requires
\[
\Phi(x) \le 0 .
\]
\end{theorem}

\begin{proof}
If \(H(x) > C\), the information required to specify the configuration exceeds system capacity, violating realizability.
\end{proof}
