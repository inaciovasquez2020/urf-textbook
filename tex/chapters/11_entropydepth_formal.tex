\chapter{Formal EntropyDepth Theory}

\section{Configuration Space}

Let \(\Omega_n\) denote the configuration space of size \(n\).

\[
H_0 = \log |\Omega_n|
\]

\section{Refinement Step}

A refinement operator \(R\) maps partitions

\[
P_{t+1} = R(P_t).
\]

\section{EntropyDepth}

\[
D = \min \{t : H(P_t)=0\}.
\]

\section{Lower Bound}

If each refinement reduces entropy by at most \(c\),

\[
D \ge \frac{H_0}{c}.
\]
