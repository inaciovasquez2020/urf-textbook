\chapter{Spectral Gap Examples}

\section{Cycle Graph}

For the cycle graph \(C_n\),

\[
\lambda_k = 2 - 2\cos\left(\frac{2\pi k}{n}\right)
\]

with spectral gap

\[
\lambda_1 = 2 - 2\cos\left(\frac{2\pi}{n}\right).
\]

\section{Path Graph}

For the path graph \(P_n\),

\[
\lambda_k = 2 - 2\cos\left(\frac{\pi k}{n+1}\right).
\]

\section{Expander Graphs}

Expander families satisfy

\[
\lambda_1 \ge \epsilon > 0
\]

independent of graph size.
