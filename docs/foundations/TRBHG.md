# Time-Reversed Black-Hole Geometry

## Status

Expository formulation only.

This note does not claim a new spacetime solution.

## Definition

\[
\mathrm{TRBHG}:=\Theta(\mathcal G_{\mathrm{BH}})
\]

where

\[
\mathcal G_{\mathrm{BH}}
=
(\mathcal M,g,\tau,\mathscr I^-,\mathscr I^+)
\]

and

\[
\Theta(\mathcal M,g,\tau,\mathscr I^-,\mathscr I^+)
=
(\mathcal M,g,-\tau,\mathscr I^+,\mathscr I^-).
\]

## Causal Regions

\[
\mathcal B^+
=
\mathcal M\setminus J^-_\tau(\mathscr I^+)
\]

\[
\mathcal H^+
=
\partial J^-_\tau(\mathscr I^+)
\]

\[
\mathcal W^-
=
\mathcal M\setminus J^+_\tau(\mathscr I^-)
\]

\[
\mathcal H^-
=
\partial J^+_\tau(\mathscr I^-)
\]

## Time-Reversal Identities

\[
J^-_{-\tau}(\mathscr I^-)
=
J^+_\tau(\mathscr I^-)
\]

\[
\Theta(\mathcal B^+)=\mathcal W^-
\]

\[
\Theta(\mathcal H^+)=\mathcal H^-.
\]

## Shadow-of-Infinity Test

\[
\operatorname{Shadow}_\tau(\mathscr I^+)
:=
\partial J^-_\tau(\mathscr I^+)
=
\mathcal H^+
\]

\[
\operatorname{Shadow}_{-\tau}(\mathscr I^-)
:=
\partial J^-_{-\tau}(\mathscr I^-)
=
\partial J^+_\tau(\mathscr I^-)
=
\mathcal H^-.
\]

Therefore

\[
\boxed{
\Theta\bigl(\operatorname{Shadow}_\tau(\mathscr I^+)\bigr)
=
\operatorname{Shadow}_{-\tau}(\mathscr I^-)
}
\]

and

\[
\boxed{
\mathrm{TRBHG}
=
\text{white-hole sector of a time-reversed black-hole spacetime}.
}
\]

## Placement

TRBHG is not a new physical object.

TRBHG is a named causal-shadow formulation of the standard white-hole/time-reversed-black-hole sector.
