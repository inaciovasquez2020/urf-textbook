# UJ Unknown-Sector Classifier Note

Status: Conditional exposition note.

Purpose:
This note records a conditional classifier for anomalous-motion reports under conservation-law discipline.

Non-claims:
- This does not prove UFO/UAP extraterrestrial origin.
- This does not prove faster-than-light travel.
- This does not prove metric engineering.
- This does not prove non-human technology.
- This does not confirm an unknown physical sector from nondetection.

Core rule:
Finite nondetection yields only bounds, not existence.

Allowed terminal classifications:
- MUNDANE_EXPLANATION
- AIR_OR_LARGE_VOLUME_FIELD_ADMISSIBLE
- NON_AIR_UNKNOWN_SECTOR_SUPPORTED
- NON_AIR_SIGNATURE_NEEDS_CONFIDENCE_UPGRADE
- NON_AIR_SECTOR_UNTESTED
- METRIC_TOPOLOGY_CANDIDATE_SUPPORTED
- ULTRA_WEAK_UNKNOWN_SECTOR_BOUND_ONLY
- FALSE_ANOMALY_OR_SENSOR_ARTIFACT
- UNCLASSIFIED

Forbidden overclaims:
- CONFIRMED_UNKNOWN_SECTOR_FROM_NONDETECTION
- CONFIRMED_METRIC_TOPOLOGY_FROM_BOUND_ONLY
- CONFIRMED_NON_HUMAN_ORIGIN_FROM_ANOMALY

Shadow-of-Infinity guard:
No branch may promote finite absence of evidence, finite nondetection, or a bound-only result into confirmed unknown-sector existence.

Minimal missing object:
Real calibrated multisensor event data.
