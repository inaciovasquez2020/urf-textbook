# URF Conceptual Flow Diagram

## Purpose
Provide a simple conceptual diagram summarizing the mechanism of the
Unified Rigidity Framework.

Local Information
↓
Bounded Information Capacity
↓
Limited Refinement Power
↓
Structural Rigidity
↓
Global Reconstruction Obstruction
## Interpretation

Local algorithms can only observe bounded neighborhoods and transmit
limited information per refinement step.

When the underlying structure contains sufficient complexity
(e.g., high cycle overlap), these constraints prevent the algorithm
from resolving all ambiguities.

This produces **rigidity**, meaning distinct global structures remain
indistinguishable under local refinement.
