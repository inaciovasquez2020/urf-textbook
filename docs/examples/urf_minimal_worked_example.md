# URF Minimal Worked Example: Cycle-Local Rigidity Obstruction

## Purpose

Provide a registry-safe minimal worked example for the rigidity / capacity mechanism by separating the false unguarded CLR claim from the guarded theorem surface.

## Setup

Let \(G=(V,E)\) be a finite simple connected graph of maximum degree \(\Delta\).
Let \(B_r(v)\) be the rooted induced radius-\(r\) ball around \(v\).
Let \(\operatorname{type}^k_r(v)\) denote the rooted \(\mathrm{FO}^k\) local type of \(v\) at radius \(r\).

## Exact Definition: Short-Cycle Rank

For \(R\ge 3\), let

\[
\mathcal C_{\le R}(G)
=
\{C\subseteq G : C \text{ is a simple cycle and } |C|\le R\}.
\]

For each \(C\in \mathcal C_{\le R}(G)\), let

\[
\chi_C\in \mathbb F_2^E
\]

be its edge-incidence vector.

Define

\[
\operatorname{COR}_R(G)
:=
\dim_{\mathbb F_2}
\operatorname{span}
\{\chi_C : C\in \mathcal C_{\le R}(G)\}.
\]

This is the short-cycle cycle-space rank generated by cycles of length at most \(R\).

## Exact Threshold Definition

The unguarded full-diversity threshold is

\[
T_{\mathrm{full}}(k,\Delta,r,R)=+\infty.
\]

Equivalently, no finite threshold depending only on \(k,\Delta,r,R\) proves

\[
\operatorname{COR}_R(G)\ge T(k,\Delta,r,R)
\Longrightarrow
\left|\{\operatorname{type}^k_r(v):v\in V\}\right|=|V|.
\]

The guarded registry-safe threshold is

\[
T_{\mathrm{guarded}}(k,\Delta,r,R)=0.
\]

## Required Obstruction Test

For \(m>2r+2\), define

\[
G_m=C_m\square C_m.
\]

Then \(G_m\) is \(4\)-regular and vertex-transitive, hence

\[
\forall u,v\in V(G_m),\quad
(B_r(u),u)\cong(B_r(v),v).
\]

Therefore

\[
\left|\{\operatorname{type}^k_r(v):v\in V(G_m)\}\right|=1.
\]

For \(R=4\), the \(m^2\) square faces span a subspace of dimension

\[
m^2-1
\]

inside \(\mathbb F_2^E\), so

\[
\operatorname{COR}_4(G_m)\ge m^2-1.
\]

Thus \(\operatorname{COR}_4(G_m)\to\infty\) while full local type diversity fails.

## Corrected CLR Surface

Define

\[
\operatorname{SymBreak}_{k,r}(G)
:=
\forall u\neq v,\
\operatorname{type}^k_r(u)\neq \operatorname{type}^k_r(v).
\]

The registry-safe guarded statement is

\[
\operatorname{COR}_R(G)\ge T_{\mathrm{guarded}}(k,\Delta,r,R)
\wedge
\operatorname{SymBreak}_{k,r}(G)
\Longrightarrow
\left|\{\operatorname{type}^k_r(v):v\in V\}\right|=|V|.
\]

Since \(T_{\mathrm{guarded}}=0\), the cycle-rank hypothesis is nonnegative bookkeeping only. The actual forcing ingredient is the explicit symmetry-breaking assumption.
