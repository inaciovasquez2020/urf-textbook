# Hard Family Specification

## Target
Construct a bounded-degree hard family
\[
\{G_n\}_{n\ge 1}
\]
satisfying
\[
H_0(n)\ge c\,n\log n
\]
for some \(c>0\), together with compatibility with the local-pattern compression bound.

## Required Properties
1. Bounded degree.
2. FO\(^k\)-local indistinguishability at controlled radius.
3. Global cycle/XOR obstruction.
4. Residual entropy lower bound
\[
H_0(n)\ge c\,n\log n.
\]

Open.
