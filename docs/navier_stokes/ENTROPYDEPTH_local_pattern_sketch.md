# Finite Local-Pattern Compression Lemma Sketch

## Target Lemma
Fix
\[
k\ge 2,\qquad \Delta\ge 3,\qquad r\ge 1.
\]
Show there exist
\[
R=R(k,\Delta,r),\qquad M=M(k,\Delta,r)<\infty
\]
such that every FO\(^k\)-definable radius-\(r\) refinement rule factors through the rooted local type map
\[
\operatorname{tp}^R_k.
\]

## Entropy Consequence
\[
\Delta H_t\le \log_2 M.
\]

## Proof Skeleton
1. Define the rooted local type space \(\mathcal T_{k,\Delta,R}\).
2. Prove \(|\mathcal T_{k,\Delta,R}|<\infty\).
3. Show every radius-\(r\) FO\(^k\) refinement depends only on \(\operatorname{tp}^R_k\).
4. Bound one-step entropy loss by \(\log_2 M\).
