\begin{example}[Exchange ECC Threshold]
\textbf{Dependencies: } D1, T2
\textbf{Status: } Illustrative

Consider an effective Hamiltonian with exchange correlation capacity (ECC)
invariant $C_{\mathrm{ECC}}$.

Let uncertainty generation in the system be measured by entropy $H$.

If
\[
H > C_{\mathrm{ECC}},
\]
then by T2 operational rigidity fails and macroscopic phase coherence
cannot be maintained.

Conversely, when
\[
H \le C_{\mathrm{ECC}},
\]
exchange-induced order persists.

Thus phase rigidity emerges as a capacity-dominated threshold phenomenon
within the URF framework.
\end{example}
