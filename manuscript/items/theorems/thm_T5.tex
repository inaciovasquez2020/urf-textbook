\begin{theorem}[Universality / Normalization]
\textbf{Dependencies: } D2, D3, L1, L2, T1
\textbf{Status: } Conditional

Assume every polynomial-time algorithm respecting bounded-local reasoning
normalizes to a transcript-limited refinement system.

Then any such algorithm operating on instances with
$H(X)=\Theta(n)$ must satisfy
\[
T \ge \Omega(n).
\]

Consequently, polynomial-time refinement under locality constraints
cannot bypass EntropyDepth lower bounds.
\end{theorem}
