\begin{lemma}[Additivity Under Conditioning]
\textbf{Dependencies: } D1, L3
\textbf{Status: } Proven

Let $T_1,\dots,T_T$ be sequential transcript variables.
Then
\[
I(X;T_1,\dots,T_T)
= \sum_{t=1}^{T} I(X;T_t \mid T_1,\dots,T_{t-1}).
\]

Each conditional term is capacity-bounded,
so global information accumulation remains limited.

Thus refinement depth reflects cumulative bounded information work.
\end{lemma}
