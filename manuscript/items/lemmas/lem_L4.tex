\begin{lemma}[Spectral Floor]
\textbf{Dependencies: } D1
\textbf{Status: } Structural

Let $\lambda_1$ denote the first nonzero eigenvalue of an admissible operator $L$.
If a coercive lower bound exists,
\[
\lambda_1 \ge 0,
\]
then no persistent non-decaying mode can survive.

In particular, violation of this bound would imply a forbidden
capacity-escaping fluctuation.
\end{lemma}
