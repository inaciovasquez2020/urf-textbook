\begin{lemma}[Non-Amplification]
\textbf{Dependencies: } D1, D2, D4
\textbf{Status: } Proven

Let $(\mathcal S, R_t)$ be a transcript-limited refinement system with transcript capacity $C$.
For any refinement sequence $(R_1,\dots,R_T)$ acting on input distribution $X$,
\[
I(X;\mathrm{Transcript}) \leq C.
\]
In particular, total extractable information cannot exceed transcript capacity.
\end{lemma}
