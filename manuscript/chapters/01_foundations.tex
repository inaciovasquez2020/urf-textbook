\chapter{Foundations}

\section{Transcript-Limited Refinement}

A refinement system is a tuple $(\mathcal S, R_t)$
where each $R_t$ produces a transcript variable $T_t$.

\section{Capacity Bound}

Each refinement step satisfies:
\[
I(X;T_t \mid T_{<t}) \le C_t.
\]

\section{Non-Amplification Principle }

Information gain cannot exceed cumulative transcript capacity.

\section{Rigidity}

If uncertainty generation exceeds capacity,
stabilization fails.

This invariant governs all subsequent domains.
