\chapter{Example: SAT and Search}

\section{Search as Refinement}

A search problem can be represented as a refinement process over candidate witnesses.  At each step, the transcript records information that removes candidates or confirms local consistency.

\begin{definition}[Search transcript]
A search transcript is a finite record of queries, responses, and eliminations generated by a search procedure.
\end{definition}

\section{Capacity View}

If each step eliminates only bounded uncertainty, then a large unresolved witness space forces many refinement steps.

\begin{example}[Finite search obstruction]
Let a witness class have entropy demand \(D\).  If each admissible query contributes at most \(C\) information, then at least \(D/C\) query steps are required.
\end{example}
